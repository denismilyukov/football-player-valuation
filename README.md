# football-player-valuation
Исследование, направленное на получение оценки стоимости футболистов и нахождение недооцененных игроков, основываясь на их игровой статистике

**Этот проект направлен на решение задачи объективного прогнозирования стоимости профессиональных футболистов.**

# Проблема
Современный профессиональный спорт переживает эпоху цифровой трансформации, и футбол как один из самых популярных видов спорта не является исключением. Мировой опыт по повсеместному сбору данных и внедрению продвинутых метрик демонстрирует, что объективная информация и математика способны выявить истинную ценность футболиста.
Однако на стыке аналитики и рынка возникает ключевой вопрос: всегда ли рыночная стоимость футболиста адекватно и корректно отражает его реальный вклад в игру? Ведь стоимость зачастую формируется под влиянием не только объективной статистики, но также и большого количества других факторов, таких как потенциал, медийность, характер, репутация, общее положение дел на рынке, маркетинговая составляющая.
Это создаёт огромный объём систематических ошибок в оценке, что ведёт к финансовым потерям клубов и неэффективному распределению ресурсов.

# Цель
*Построить модель машинного обучения, которая сможет давать объективную оценку игрокам на основании их игровых характеристик, возраста и чемпионата. Соответствующие данные были собраны за сезон 2024/2025.*

# Схема достижения цели

1. Сбор данных (реализаия алгоритмов парсинга данных из авторитетных источников  [FBref](https://fbref.com/) и [Transfermarkt](https://www.transfermarkt.world/))
2. Предобработка данных (ручная обработка ошибок парсинга, интеграция данных из разных источников, обработка дубликатов, пропусков, слияние полученных таблиц в общую)
3. Построение моделей ML (в качестве бейслайна выбрана линейная регрессия, основная модель - xgboosting)

# Как использовать (инструкция для Windows)

1. Клонируйте репозиторий
   ```git clone https://github.com/denismilyukov/football-player-valuation.git```
2. Перейдите в репозиторий
   ```cd football-player-valuation```
3. Установите зависимости
   ```pip install -r requirements.txt```
4. Перейдите в папку с ноутбуками
   ```cd notebooks```
5. Запустите Jupyter Notebook
   ```jupyter-notebook```
